Nota: como parámetros puede ingresar los directorios que desee para ser analizados. Cada directorio debe ir separado por un espacio. En caso de no ingresar ningún directorio o parámetro el programa se leerá a sí mismo por defecto. En caso de ingresar un directorio que no sea valido el sistema le indicara que ha ingresado un valor invalido. Si ingresa un directorio que no contenga ningún archivo .java el programa mostrara un resultado vacío en la tabla de resultados.

Este es un ejemplo de cómo indicarle al programa un directorio:

java -cp target/Psp1SonarISM-1.0-SNAPSHOT.jar co.com.ism.controller.ControllerDemo /Users/ian/Documents/workspaceLuna/psp0
